## APRESENTAÇÃO

- Essa aplicação foi desenvolvida como parte do processo de seleção para a posição de Desenvolvedor(a) Full Stack Sênior especializado em C# e MVC através da FloeHire, consultoria de recrutamento e seleção

## COMO EXECUTAR O PROJETO

- Clonar o repositório;
- Executar o comando "dotnet run" nos diretórios "Enilton.Gateway" e "Enilton.Api"
- Executar o comando "ng serve" no diretório "Enilton.Store"
- A persistência está sendo realizada em bancos na nuvem que criei com objetivo unico de utilizar para este processo.
